---
name: Objetivo del proyecto Análisis de Ventas Luctron
description: Construir dashboard de ventas de Luctron y diseñar estrategia para alcanzar un objetivo mensual de ARS 300M / USD 215k.
type: project
---

El proyecto "Analisis de Ventas Luctron" tiene dos fases:

1. **Dashboard analítico**: HTML interactivo con datos embebidos para analizar las ventas año a año y mes a mes (foco inicial: facturación total y ventas por vendedor).
2. **Estrategia comercial**: usar las conclusiones del dashboard para armar un plan que lleve la facturación al objetivo.

**Objetivo de venta (definido 2026-04-24):** ARS 300.000.000 mensuales ≈ USD 215.000 mensuales al TC del día (~1.395 ARS/USD).

**Why:** Nacho ya tiene un número en mente; no hay que estimarlo a partir de los datos. Las comparaciones y proyecciones tienen que medirse contra este target mensual.

**How to apply:** Cuando muestre KPIs, comparativas o proyecciones, incluir siempre el % de cumplimiento del objetivo mensual (en USD por defecto, ya que la inflación argentina distorsiona el ARS). Usar USD constantes para comparativas YoY a lo largo de 6+ años de historia.
