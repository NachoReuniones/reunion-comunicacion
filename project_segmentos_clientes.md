Aunque el **vertical principal** de Luctron son las **estaciones de servicio** (donde introdujeron el formato lineal de luminaria para aleros líquidos/GNC), la base de clientes muestra presencia consolidada en varios segmentos:

## Segmentos con clientes históricos
- **Farmacéutico / retail farma:** Farmacity, Farmalife, Farmacia La Esquina, Farmacia Belgrano, Farmacias Chester
- **Energía / combustibles:** Raizen (Shell), Pan American Energy, Petrolera Cruz del Sur — además de las grandes petroleras que son clientes "core" del vertical principal
- **Logística:** Andreani (top cliente 2025)
- **Industria / salud / distribución farma:** Suizo Argentina, Mediven, Servicios para la Salud
- **Agro / alimenticio:** Unión Agrícola Avellaneda, Agropez, Conarpesa (pesca)
- **Construcción / real estate:** Urbanizar, Monge Construcciones, Miller Building
- **Industria metalúrgica:** Metalúrgica El Pampa, Conintec SRL

## Asignación de verticales por vendedor (mayo 2026)

| Vertical | Vendedor responsable |
|----------|----------------------|
| Farmacias | **Facundo** (exclusividad) |
| Logística | **Facundo** (exclusividad) |
| Industria | **Facundo** principal, **Lucas** apoyo |
| Estaciones de servicio | **Federico** (~95%) + **Lucas** (apoyo, alivianar carga) |
| Áreas deportivas | **Lucas** |
| Oficinas | **Lucas** |
| Universidades | **Lucas** |
| Otros rubros varios | **Lucas** |
| Negocios "puente" (estacioneros que traen otro rubro) | **Federico** |
| Cuentas históricas / dirección | **INX2 / Nacho** |

Esto significa que **el desempeño por vendedor en el dashboard equivale aproximadamente al desempeño del vertical correspondiente**. Por ejemplo, el crecimiento de Facundo (+46% YoY 2025) es esencialmente el crecimiento combinado de farma + logística + industria.

**Why:** Aunque el branding y diferencial técnico está fuertemente vinculado a estaciones de servicio, la realidad comercial es multi-vertical. Importante para no encajonar el análisis estratégico.

**How to apply:** Al hablar de segmentos, no asumir que todo es estaciones de servicio. Cruzar análisis de clientes con estos verticales puede revelar oportunidades (ej. consolidar farma como segundo vertical estratégico).
